# ✊📄✂️ Rock Paper Scissors Game

A simple and interactive **Rock Paper Scissors** game built using **HTML5**, **CSS3**, and **JavaScript**. The project demonstrates fundamental web development concepts, including user interaction, game logic implementation, and dynamic DOM manipulation.

🔗 **Live Demo:** https://aneeshkumaryadav.github.io/RockPaperScissor/

---

# 📖 Project Overview

This project is a browser-based implementation of the classic **Rock Paper Scissors** game. Player can compete against the computer, which generates random choices for each round.

The game provides instant feedback, keeps track of results, and offers an engaging way to practice JavaScript fundamentals such as event handling, conditional logic, and DOM updates.

This project was developed to strengthen front-end development skills and gain hands-on experience in creating interactive web applications.

# 🎮 How to Play

1. Choose **Rock**, **Paper**, or **Scissors**.
2. The computer will automatically make its choice.
3. The game compares both choices.
4. The winner of the round is displayed instantly.
5. Continue playing and track your score.

### Rules

* Rock beats Scissors
* Scissors beats Paper
* Paper beats Rock
* Same choices result in a Draw
